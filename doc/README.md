## Quick Reference

- **Level cap:** 50
- **Classes:** Warrior, Paladin, Rogue, Ranger, Mage, Healer, Sage
- **Acts:** Dark Forest (Lv 1–12) → Desert Saharrrra (Lv 13–35) → Abyssal Rift (Lv 36–50)
- **Skill points per level:** 1
- **Attribute points per level:** 5
- **Skill/Attribute reset cost:** 3,000 gold each
- **Crit damage:** 2× (Ranger Hunter's Instinct can boost this further)
- **Dodge/Crit/Block cap:** 75%
- **Damage reduction cap:** 50%
